# Getting this code to work

Installation:

1. Install Xcode


2. Install llvm compiler


```bash
brew install llvm
```

3. Install OpenMP

```bash
brew install libomp
```

4. Clone the repo

```
git clone https://github.com/larsgeb/msl-tutorial.git
```
